# Quantum Nexus

A text based adventure game where the laws of quantum mechanics are the puzzle.
You have 25 turns to escape a simulation that has become aware of you.

The same quantum rules drive three different worlds — pick your choice, then
talk your way out using natural language commands.

---

## The Premise

You are trapped in a simulation. Something inside it is **measuring** you —
and in this universe, being observed has consequences. Objects exist in
superposition until they are looked at. Two items entangled with each other
respond as one. The wrong move collapses the wavefunction, and you with it.

Twenty five turns. One way out. The Observer is patient.

---

## Scenarios

The mechanics are identical across all three. The world, tone, and visualizer
change.

| Scenario | Setting | Mood |
|---|---|---|
| **QLAB-7** | A quantum research facility, 2157 | Cold sci-fi. *Annihilation* meets a physics textbook. |
| **RUNNER-2050** | A megacity drowning in perpetual rain | Neon noir. Corporate dystopia. Every reflection is wrong. Inspired from Blade Runner 2049. |
| **TET-7** | A surface outpost on a depopulated Earth | Post apocalyptic minimalism. Dust, wind, and distant drones. |

Pick one from the launch screen. The puzzle is the same. The story isn't.

---

## Features

- **AI-narrated prose** — every action gets fresh narrative generated by an
  LLM, styled to match the chosen scenario's voice. Each scenario has its own
  forbidden words, atmosphere words, and narrator personality.
- **Built-in fallback narrator** — playable without an API key; the game uses
  scripted text instead of AI prose.
- **Scenario-aware quantum field visualizer** — a side panel renders an
  animated particle field that re-skins itself per scenario:
  - QLAB-7 → orderly green particles, clean lattice
  - RUNNER-2050 → cinematic neon: silhouetted figure made of particles, glyph
    rain, wet pavement reflection, slow color drift between pink and violet
  - TET-7 → slow swirling sepia dust with periodic drone sweep beam
- **Topology map with fog of war** — schematic node graph above the visualizer
  showing rooms you've explored, with dim "???" hints for adjacent unvisited
  rooms. Per-scenario room labels. Reveals connectivity without spoiling
  puzzles.
- **Reactive effects** — observer status changes the palette, item pickups
  send out a ring burst, room changes trigger a wavefunction collapse, the
  field goes critical when you're running out of turns.
- **Background music** — toggleable ambient audio with a one click button in
  the header.
- **Natural language input** — type the way you'd think it.
  *"examine the door," "take the mirror key," "go north,"
  "use quantum key on core stabilizer."*

## Setup

### Requirements
- Python 3.9+
- pip

### Install

```bash
git clone https://github.com/Hitha99/TheGame.git
cd TheGame
pip install -r requirements.txt
```

### (Optional) AI narration

Quantum Nexus uses the TAMU AI Chat API for AI-generated prose. You can play
without it, but the prose will be richer with a key.

- Click the **⚙ API KEY** button in the top right of the game.
- Paste your key. It's stored only in your browser's local storage and sent
  directly from your browser to your local Flask server and never to any third
  party.
- Leave it blank and the game uses fallback narration.

### Run

```bash
python3 app.py
```

Open `http://127.0.0.1:5000` in your browser.

## How to Play

You have 25 turns. Type what you want to do in plain English — the parser
handles natural phrasing.

### Commands the parser understands

```
look                              # describe the current room
examine <object>                  # inspect something
take <object>                     # pick up an item
drop <object>                     # drop an item from inventory
go <direction>                    # north, south, east, west, across
use <object> on <target>          # combine / activate
hide                              # try to break the Observer's gaze
```

### Reading the status bar

```
◆ Rooftop Access  │  TURN 0/25  │  OBSERVER: WATCHING  │  INV: —  │  EXITS: north · east
```

| Stat | What it means |
|---|---|
| Room | Where you are. Names change per scenario. |
| TURN | How many turns you've used / your 25 turn budget. Goes red at ≤ 5. |
| OBSERVER | `WATCHING` (amber) → `DISTRACTED` (green) → `DEFEATED`. |
| INV | What you're carrying. |
| EXITS | Directions you can try to move. |

### Reading the map

The topology map (top right panel) shows your progress through the simulation:

- **Bright glowing dot** — your current room
- **Solid dots with names** — rooms you've explored
- **"???" outlined dots** — adjacent rooms you haven't entered yet (you know
  an exit exists, but not what's there)
- **Dashed edges** — exits to unexplored territory
- **Solid edges** — connections between rooms you've already walked

The map persists between page refreshes for the same scenario, but resets
when you click Restart.

### Tips without spoilers

- **The Observer is the win condition, not just a threat.** Pay attention
  to what changes its state.
- **Entangled objects share fate.** Touching one affects the other.
- **Time pressure is real.** The narrative will not warn you that you're
  running out of turns. The status bar will.
